This is a grid-based, phase-based tactical simulator created as a portfolio project.

The project takes inspiration from grid-based strategy games such as Shogi, Chess,
and Fire Emblem, with a focus on clean architecture and testable core logic.

## Technologies
- C# (.NET)
- NUnit
- Git / GitHub

## Planned Features
- Tactical simulation core logic
- Deterministic and testable systems
- Unity integration
